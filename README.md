## Description

Pokemon API.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Start DB

```bash
$ docker-compose up -d
```

## Start Seed

```bash
http://localhost:3001/api/v2/seed
```

## Stack
* NestJS
* MongoDB
* Docker
* Docker Compose
