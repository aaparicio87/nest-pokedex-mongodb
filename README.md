## Description

Pokemon API.

## Installation

```bash
$ pnpm install
```

## Start DB

```bash
$ docker-compose up -d
```

## Config environment variables

Copy the file __.env.sample__ to __.env__ 
and edit it with your own values.

## Running the app in development mode

```bash
$ pnpm run start:dev
```

## Start Seed

```bash
http://localhost:3001/api/v2/seed
```

## Stack
* NestJS
* MongoDB
* Docker
* Docker Compose
* TypeScript
* Joi
